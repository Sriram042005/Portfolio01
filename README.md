### Prerequisites
- Node.js installed (v16 or higher recommended)

### 1. Navigate to Project Directory
Open your terminal (Command Prompt, PowerShell, or VS Code Terminal) and go to the project folder:
```bash
cd creative-deity-portfolio
```

### 2. Install Dependencies
If this is your first time running it, install the required packages:
```bash
npm install
```

### 3. Start Development Server
Run this command to start the local server:
```bash
npm run dev
```
> You will see a URL like `http://localhost:5173/`. Ctrl+Click it to open in your browser.

### 4. Build for Production (Optional)
To generate the final files for hosting (e.g., GitHub Pages, Vercel):
```bash
npm run build
```
The output will be in the `dist/` folder.
